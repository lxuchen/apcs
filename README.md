# Java Learning Workspace

This workspace is designed for learning Java and preparing for AP Computer Science A.

## Structure

- `src/`  
  Contains Java source files. Each `.java` file can be run independently in VS Code.

- `notes/`  
  Markdown notes for each AP CS topic.

- `playground/`  
  A scratch area for experimenting with Java code (`Scratch.java`).

- `.vscode/`  
  Pre-configured settings and debug options for the Java environment.

## How to Use

1. Open this folder in VS Code.
2. Install the recommended extensions (Extension Pack for Java).
3. Open any Java file in the `src/` folder.
4. Click **Run** or press `F5` to execute.
5. Write notes and examples in the `notes/` directory.

Enjoy learning Java!
